# Quick Start (Mac)

## 1. Open in Godot 4.3

1. Install **Godot 4.3** (Standard): https://godotengine.org/download/macos/
2. Open Godot → **Import**
3. Select: `jukai-no-yami/project.godot`
4. Click **Import & Edit**

If Godot asks **discard vs keep local changes** → choose **Keep local changes**.

## 2. Run

Press **F5** (Play). Main menu → **Start Game**.

## 3. If you see old errors

Delete the cache folder (Godot recreates it):

```bash
rm -rf /Users/mireikaterina/Desktop/Jukei/jukai-no-yami/.godot
```

Then reopen the project in Godot.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| F | Flashlight |
| E | Interact |
| Shift | Sprint |
| Esc | Pause |

Settings: Main menu → **設定 / Settings** → Graphics **Medium** (best balance on MacBook).

**v2 visual style:** Chillas Art / Fears to Fathom — dark fog, flashlight-only lighting, no VHS scanlines.
